# gentoo--
纯小白傻瓜式安装😃

1.一个电脑
2.会打字的手
3.ubuntu镜像
4.一个U盘
5.下载一个rufus，或者是ventoy（直接制作不想多说😃）


开始安装！

从U盘启动。
开机选择U盘启动，把U盘插入USB口，开机疯狂按Delete进入BIOS，选择USB储存介质启动，进入刚刚制作的启动U盘。

联通网络。
Ubuntu联网大家都会吧应该。

硬盘分区。
首先打开终端，输入 sudo -i ，输入lsblk查看当前的电脑分区。这里使用cfdisk进行分区，输入 cfdisk /dev/sda 进行分区。
